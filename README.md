## Ответы на вопросы

**Что делает fetch?**  
`fetch` выполняет HTTP-запрос к серверу и возвращает Promise с объектом ответа. Его используют для получения данных с API, файлов или других ресурсов.

**Зачем нужны async/await?**  
`async/await` упрощают работу с асинхронным кодом. `async` делает функцию асинхронной и возвращает Promise, а `await` ставит выполнение на паузу до завершения Promise, что позволяет писать асинхронный код в стиле синхронного.

**Как работает localStorage?**  
`localStorage` — это хранилище в браузере, которое сохраняет данные в виде ключ-значение. Данные сохраняются даже после перезагрузки страницы. Хранить можно только строки, объекты нужно сериализовать через `JSON.stringify`.

**Где помог ИИ, а где пришлось разбираться вручную?**  
ИИ помог с генерацией асинхронного кода для загрузки отзывов и картинок, обработкой состояния "Загрузка..." и логикой переключателя темы с localStorage.  
Ручная работа потребовалась для интеграции кода на страницу, настройки аккордеона, галереи с фильтрами и кнопки "Наверх".
